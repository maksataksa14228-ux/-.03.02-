# Практическая работа по управлению проектами

## Цель
Изучение Git и управления проектами в рамках практической работы.

## Структура проекта
my-project/
|----src/ #исходный код проекта
|----README.md #документация
|----.gitignore #исключённые файлы


## Запуск проекта
1. Убедитесь, что установлен Python 3.7+
2. Выполните в терминале:
   python src/main.py
