# Настройка окружения разработки

## Требования
- Операционная система: Windows 10/11, macOS, Linux
- VS Code (последняя версия) — [скачать](https://code.visualstudio.com/)
- Git — [скачать](https://git-scm.com/)

## Быстрая настройка за 5 минут

### Шаг 1: Клонировать репозиторий
\\\ash
git clone https://github.com/ваш_аккаунт/my-project.git
cd my-project
\\\

### Шаг 2: Установить расширения VS Code

**Windows (PowerShell):**
\\\powershell
.\\.dev-env\\setup.ps1
\\\

**macOS/Linux:**
\\\ash
chmod +x .dev-env/setup.sh
./.dev-env/setup.sh
\\\

### Шаг 3: Применить настройки VS Code
1. Откройте VS Code
2. Нажмите \Ctrl + Shift + P\ (или \Cmd + Shift + P\ на macOS)
3. Введите: \Preferences: Open Settings (JSON)\
4. Скопируйте содержимое файла \.dev-env/settings.json\ в открывшийся файл
5. Сохраните (Ctrl+S)

### Шаг 4: (Опционально) Применить горячие клавиши
1. \Ctrl + Shift + P\ → \Preferences: Open Keyboard Shortcuts (JSON)\
2. Скопируйте содержимое \.dev-env/keybindings.json\

## Проверка работоспособности
- Откройте любой файл и нажмите \Shift + Alt + F\ — должно сработать форматирование
- Посмотрите список расширений (Ctrl+Shift+X) — должны быть установлены все плагины

- Персональные токены доступа
